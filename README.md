# HERR Wellness

A wellness website for women focused on fitness, mindfulness, and community. HERR (Healthy Experiences Rest and Recovery) helps women live well inside and out.

## Pages

### Public Pages
- **index.html** — Homepage with hero section, philosophy cards, experiences grid, instructors slider, self-care checklist
- **about.html** — About page
- **events.html** — Events and updates page
- **contact.html** — Contact page with form

### Private Vendor Pages (unlisted)
- **vendor-catering.html** — Catering partner invitation for the HERR launch event
- **vendor.html** — Generic vendor partner invitation for other service providers

## Features

### Homepage
- Animated hero section with floating accent shapes
- Octagon image gallery with smooth animations
- Philosophy cards (Mind, Body, Spirit)
- HERR Experiences grid
- Instructors section with slider (Kitt & Bella)
- Video toggle for instructor intros
- Self-care micro-checklist with affirmations
- Spin class photo slider

### Design
- Color palette: Purple (#9E41B4), Pink (#E5769B), Green (#2e8b57), Cream backgrounds
- Responsive design for mobile, tablet, and desktop
- Smooth animations and transitions
- Modern, wellness-focused aesthetic

### Forms
- Email signup in welcome section
- Email signup in footer
- Contact form (via Formspree)
- Vendor inquiry forms (via Formspree)
